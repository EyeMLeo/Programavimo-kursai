  • element / tgo vardasz
  • .CLASS pagal tagui suteikta klase
  • #ID pagal tagui suteikta ID
  • .class1.class2 / tagui suteiktos 2 ar daugiau klases
  • .class3 .class4 tagui suteikia stilius,jeigu jis turi class4 klase ir yra class3 palikuonis
  • * / parenka viska
  • element1, element2 / isvardintus ir kableliu atskirtus elementus
  • element1 > element2 / element 2 jkurio tevinis elementas yra element1
  • a:link / nelankytos nuoruodos
  • A:visited / lankytos nuoruodos
  • a:active / aktyvi nuoruoda (paspaudimo momentas)
  • element:hover / elementas ant kurio yra uzvesta pele
